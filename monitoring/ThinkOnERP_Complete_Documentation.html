🔐 Authentication & Authorization

Authentication System

ThinkOnERP uses JWT (JSON Web Token) based authentication with refresh token support. The system supports two types of users:

  • Regular Users: Company employees with role-based access
  • Super Admins: System administrators with full access

User Login

POST /api/auth/login

Request Body:

{
  "userName": "string",
  "password": "string"
}

Success Response (200 OK):

{
  "success": true,
  "message": "Authentication successful",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "refreshToken": "refresh_token_string",
    "expiresAt": "2024-12-31T23:59:59Z",
    "refreshTokenExpiresAt": "2025-01-07T23:59:59Z",
    "userId": 123,
    "userName": "john.doe",
    "role": "User",
    "companyId": 1,
    "branchId": 5
  },
  "statusCode": 200
}

Super Admin Login

POST /api/auth/superadmin/login

Same request/response format as user login, but for super admin accounts.

Token Refresh

POST /api/auth/refresh

Request Body:

{
  "refreshToken": "refresh_token_string"
}

👨‍💼 Super Admin Management

Dashboard

GET /api/superadmins/dashboard

Provides system-wide overview including metrics, alerts, recent activity, and system health monitoring.

CRUD Operations

GET /api/superadmins

List all super admins

GET /api/superadmins/{id}

Get super admin details

POST /api/superadmins

Create new super admin

PUT /api/superadmins/{id}

Update super admin

DELETE /api/superadmins/{id}

Delete super admin (soft delete)

Password Management

PUT /api/superadmins/{id}/change-password

Change password (requires current password)

POST /api/superadmins/{id}/reset-password

Reset password (generates temporary password)

🏢 Company Management

Company List

GET /api/companies

Retrieves all active companies with logos (Base64 encoded).

Create Company

POST /api/companies

Creates a new company with default branch in a single API call. Supports logo upload via Base64.

Request Body Example:

{
  "companyNameAr": "شركة الاختبار",
  "companyNameEn": "Test Company",
  "companyCode": "TC001",
  "legalNameAr": "شركة الاختبار للتجارة",
  "legalNameEn": "Test Company for Trading",
  "taxNumber": "123456789012345",
  "countryId": 1,
  "currId": 1,
  "companyLogoBase64": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA...",
  "branchLogoBase64": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA...",
  "branchNameAr": "الفرع الرئيسي",
  "branchNameEn": "Main Branch",
  "branchDefaultLang": "ar",
  "branchBaseCurrencyId": 1,
  "branchRoundingRules": 2,
  "branchPhone": "+966112345678",
  "branchMobile": "+966501234567",
  "branchEmail": "main@testcompany.com"
}

Update Company

PUT /api/companies/{id}

Delete Company

DELETE /api/companies/{id}

Logo Management

Upload Requirements:

  • Max file size: 2MB
  • Allowed formats: PNG, JPG, JPEG
  • Recommended dimensions: 200x200px
  • Encoding: Base64 with data URI prefix
